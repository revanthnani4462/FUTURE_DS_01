# Business Sales Performance Dashboard

## Objective

Analyze business sales data to identify revenue trends, top-selling products, category performance, and regional sales insights.

## Tools Used

* Power BI
* Excel

## Dashboard Features

* Total Sales KPI
* Total Profit KPI
* Sales Trend Analysis
* Top Selling Products
* Regional Sales Performance
* Category-wise Sales Analysis
* Interactive Filters

## Key Insights

* Technology category generated highest sales.
* West region showed strongest performance.
* Sales trends changed over time.
* Certain products generated significantly higher revenue.

## Recommendations

* Focus marketing on profitable products.
* Improve sales strategies in low-performing regions.
* Increase focus on high-performing categories.
